# Africaviwo — Nos racines. Nos valeurs. Notre futur.

Africaviwo est un projet qui célèbre les valeurs, les voix et les futurs de l'Afrique. Une exploration moderne et sincère des cultures, valeurs et élans qui font l'Afrique d'aujourd'hui.

## Stack technique

- **Framework** : React 18 + TypeScript
- **Build** : Vite 5
- **UI** : TailwindCSS + shadcn/ui
- **Contenu** : MDX
- **Routing** : react-router-dom v6

## Commandes

```bash
npm run dev      # Développement
npm run build    # Production
npm run preview  # Prévisualisation
```
